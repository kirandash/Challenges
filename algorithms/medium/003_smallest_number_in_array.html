Find Smallest Number in an Array

Create a function which takes an Array of numbers as input and returns the smallest number from array
Ex: 
                searchSmallestNumber([123, 345, 888, 4]) ➞ 4
                searchSmallestNumber([-34, 55, -56, 434]) ➞ -56
                searchSmallestNumber([0.234, 0.43, 0.76, 0.834]) ➞ 0.234
