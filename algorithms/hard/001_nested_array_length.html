Length of a Nested Array

Create a function which will calculate the length of a Nested Array
Ex: 
                lengthOfArray(['a', ['b', 'c', 'd']]) ➞ 4
                lengthOfArray(['a', ['b', ['c', ['d', 'e']]]]) ➞ 5
                lengthOfArray(['a', ['b'], 'c', 'd']) ➞ 4
                lengthOfArray(['a', ['b', 'c', 'd'], 'e']) ➞ 5
