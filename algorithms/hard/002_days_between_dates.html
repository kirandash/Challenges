How Many Days Between Two Dates

Create a function which will take two dates as input and returns the number of days between them
Ex: 
                findDays("January 10, 2020", "January 16, 2020") ➞ 6
                findDays("December 29, 2019", "January 1, 2020") ➞ 3
                findDays("March 1, 2020", "March 2, 2020") ➞ 1
